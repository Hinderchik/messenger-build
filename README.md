# Messenger Builds

Автоматическая сборка APK и EXE для мессенджера.

## 📥 Скачать последнюю версию

- [Android APK](https://github.com/Hinderchik/messenger-build/releases/latest)
- [Windows EXE](https://github.com/Hinderchik/messenger-build/releases/latest)

## 🔨 Сборка

Каждый пуш в main триггерит сборку:
- Android APK (Capacitor)
- Windows EXE (Electron)

## 📱 Установка

### Android
1. Скачай APK из Releases
2. Разреши установку из неизвестных источников
3. Установи

### Windows
1. Скачай Setup.exe из Releases
2. Запусти установку
3. Готово!

## 🚀 Технологии

- Веб-приложение: HTML/CSS/JS + Supabase
- Android: Capacitor
- Windows: Electron
